# Multi-Tool Telegram Bot

## Features
- 📱 Facebook Account Checker (Demo)
- 🔐 2FA Code Generator
- 📧 Temporary Email Service
- 💰 Credit System
- 📊 Referral Program
- 👥 Admin Panel
- 💸 Withdrawal System
- 🆘 Support Ticket System

## Deployment on Railway

1. Fork this repository
2. Create new project on Railway
3. Add environment variable:
   - `BOT_TOKEN` = Your bot token
   - `ADMIN_IDS` = Your Telegram user ID
4. Deploy!

## Commands
- /start - Start bot
- /menu - Main menu
- /myid - Get user ID
- /credits - Check balance
- /refer - Get referral link
- /withdraw - Withdraw credits
- /help - Help guide

## Credits System
- New users: 10 free credits
- Referral: 5 credits each
- Facebook check: 1-2 credits
